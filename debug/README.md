# Debug Directory

This directory contains debug artifacts and is ignored by git.

## Structure

- `screenshots/` - Debug screenshots and visual test artifacts
- `test-artifacts/` - Test output files, logs, and debugging data

## Usage

```bash
# Save debug screenshots here
debug/screenshots/component-issue.png

# Save test artifacts here  
debug/test-artifacts/api-response.json
```

**Note**: All contents of this directory are ignored by git and should not be committed.
